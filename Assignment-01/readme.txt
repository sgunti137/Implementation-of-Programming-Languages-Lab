Environment:
  Operating System: Windows
  Compiler: MinGW
  g++ version: 9.2.0
Also working fine with Ubuntu 20.04 LTS with same version of g++.

Execution:
  keep both source code and input.txt in the same folder and,
  execute the command "g++ 190101080_Assign01.cpp -o main",
  now run the command "./main"

File Description:
  190101080_Assign01.cpp : CPP file which contains the code for 2 pass assembler.
  input.txt : The text file which has input assembly language code.
  intermediate.txt : The text file which will be created after the execution of pass 1.
  listing.txt : This text file will be created after the execution of pass 2.
  output.o : Final object code that will be generated after pass2.
  objectWithCarets.txt : The text file which contains the final object code with delimiters for readability of object code.
